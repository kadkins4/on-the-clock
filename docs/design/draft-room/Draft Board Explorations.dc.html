THE MOCK DRAFT BOARD

Four board concepts shown on desktop and phone, plus six player-card treatments — all running on one shared live mock-draft engine.
VINTAGE SERIES · EST. 2026
SNAKE · LINEAR · AUCTION
6–14 TEAMS
ASSUMPTIONS
15-round rosters, single-QB league. Sample player pool ordered by 2026 ADP. Position color is the primary at-a-glance layer — six hues, one chroma. Auction assumes $200 budgets. The pick timer is decorative in this mock.
DESIGN REASONING
Vintage card-and-felt direction: cream paper, dark ink, flat sticker tints on a dark board. Type is Alfa Slab One (display) + Barlow Condensed (names, UI) + Courier Prime (data). All four boards run the same engine so you can compare them apples-to-apples mid-draft.
HOW TO REVIEW
Use the dark control bar to resize the league (6–14), switch snake / linear / auction, and run the pick simulator. Tap any filled cell or row to open its player card. Mix & match: pick one board concept + one card treatment.
